## 2.B树

### 2.1 B树基础

1.   外部节点

     通俗来讲就是数值为NULL的节点

     计算**B树的高度**（h）需要计算外部节点

2.   内部节点

     | **属性**                | **下限 (Minimum)**      | **上限 (Maximum)** | **备注**                       |
     | ----------------------- | ----------------------- | ------------------ | ------------------------------ |
     | **分支数量 (Children)** | $\lceil b/2 \rceil$     | $b$                | 保证树的填充率不低于 50%       |
     | **关键字数量 (Keys)**   | $\lceil b/2 \rceil - 1$ | $b - 1$            | 每个节点包含的关键字比分支少 1 |

     

     也可以叫b阶B树为$(\lceil b/2 \rceil,b)$树

     >   根节点没有下限限制，因为可以只有根节点
     
     >   m阶B树，是指分支数量最多为m，这样更好记忆 
     



#### 高度和节点的关系

设$n$为关键字总数，$m$阶B树

$$
\log_m (n+1) \le h \le \log_{\lceil m/2 \rceil} (\frac{n+1}{2}) + 1
$$


>   注意IO操作不需要最后搜索外部节点，所以算出了高度后需要-1

## 3. 红黑树

-   树根必为黑色 

-   外部节点均为黑色 

-   红节点的孩子必为黑色 

-   从任一节点x到其每个后代nullptr，沿途经过的（真）黑后代一样多

    这个黑后代的数目（不计x自身），称作x的<span style="color:#FF0000;">黑高度</span>

>   注意**邓书**中高级搜索树高度的计算：BST不计算外部节点，B树需要计算外部节点，红黑树模糊，书上和ppt不一样

### 3.2 删除

#### 3.2.0 一般情况：其一为红

直接桃代李僵，拿最**前驱或者后继**去替代需要删除的节点



#### 3.2.1 特殊情况

`s`是 `sister`节点

`x`是需要删除的节点

`p`是父节点 

| 情况   | 判定                                 | 染色次数 | 旋转次数 | 结果                             |
| ------ | ------------------------------------ | -------- | -------- | -------------------------------- |
| **1**  | 兄弟 `s` 为黑，且 `s` 至少一个红孩子 | 3        | 1~2      | 一蹴而就                         |
| **2R** | `s` 为黑，两孩均黑，`x p` 为红       | 2        | 0        | 一蹴而就                         |
| **2B** | `s` 为黑，两孩均黑，`x p` 为黑       | 1        | 0        | 可能传播 `O(log n)` 层           |
| **3**  | 兄弟 `s` 为红（孩子均黑）            | 2        | 1        | 必随即转入 **BB-1** 或 **BB-2R** |

可能的转换：

-   2B -> 2R
    2R -> 1
    3 -> 1 或 2R

不可能的转换：

-   2B -> 1
    2B -> 3



## 4. 跳转表

#### 4.1 主要思想

1.   节点随机生长
2.   横向为层，纵向成塔



#### 4.2 层高上界（w.h.p.）

$n $ 个节点，跳转表高度超过 $c\log n $ 的概率：
$$
P[h > c\log n] = n \cdot \left(\frac{1}{2}\right)^{c\log n} = n \cdot n^{-c} = n^{1-c}
$$
取 $c=3 $，概率仅 $\frac{1}{n^2} $，**以极高概率（w.h.p.）层高为 $O(\log n) $**。



| 操作   | 期望         | 最坏          |
| ------ | ------------ | ------------- |
| search | $O(\log n) $ | $O(n) $       |
| insert | $O(\log n) $ | $O(n) $       |
| remove | $O(\log n) $ | $O(n) $       |
| 空间   | $O(n) $      | $O(n\log n) $ |
