# 常见时间复杂度符号定义

设 $f(n), g(n)$ 为非负函数（当 $n$ 足够大时)，常见的三种渐近符号定义如下：

1. **大 O（上界）**：$f(n) = O(g(n))$
    - 含义：$f(n)$ 的增长速度不超过 $g(n)$ 的常数倍。
    - 定义：存在常数 $c > 0$、$n_0$，使得对所有 $n \ge n_0$，有
       $$0 \le f(n) \le c\,g(n)$$

2. **大 $\Omega$（下界）**：$f(n) = \Omega(g(n))$
    - 含义：$f(n)$ 的增长速度至少是 $g(n)$ 的常数倍。
    - 定义：存在常数 $c > 0$、$n_0$，使得对所有 $n \ge n_0$，有
       $$0 \le c\,g(n) \le f(n)$$

3. **大 $\Theta$（紧界）**：$f(n) = \Theta(g(n))$
    - 含义：$f(n)$ 与 $g(n)$ 同阶，即同时有上界和下界。
    - 定义：存在常数 $c_1, c_2 > 0$、$n_0$，使得对所有 $n \ge n_0$，有
       $$0 \le c_1 g(n) \le f(n) \le c_2 g(n)$$



# 主定理 (Master Theorem)

主定理用于求解分治算法中常见的递归方程：
$$T(n) = aT\left(\frac{n}{b}\right) + f(n)$$  或者 $$T(n) = aT\left(\frac{n}{b}\right) + g(n)$$

## 2. 注意事项（失效场景）

主定理并非万能，以下情况不可直接使用：
1. **$a$ 不是常数**：例如 $T(n) = n T(n/2) + f(n)$。
2. **$f(n)$ 与 $n^{\log_b a}$ 之间不是多项式差异**：
   - 例如 $T(n) = 2T(n/2) + n \log n$。
   - 虽然 $f(n)$ 看起来比 $n$ 大，但它只比 $n$ 大了 $\log$ 倍，不是 $n^\epsilon$ 倍，需使用情况2的泛化或递归树法。
3. **正则条件不满足**：在情况3中，如果无法证明 $a f(n/b) \le c f(n)$，则结论不成立。
