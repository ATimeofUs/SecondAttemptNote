## 进程互斥操作

注意： p = wait，v = signal

### PV操作函数

```c++
void P(Semaphore S) {
    S --;
    if (S < 0) {
        // 将当前进程加入等待队列
        block(queue);
    }
}

void V(Semaphore S) {
    S ++;
    if (S <= 0) {
        // 从等待队列中唤醒一个进程
        wakeup(queue);
    }
}
```

如何理解 S 的数字含义？

S >= 0 ：有多少可以用的资源

S < 0：S被分配完了，有多少人在等

### 生产者消费者问题

```c++
semaphore mutex = 1, empty = n, full = 0;

void Produce(){
	while (1){
        P(empty);
        P(mutex);
        生成;
        V(mutex);
        V(full);
    }
}

void Consume(){
    while (1){
        P(full);
        P(mutex);
        取出;
        V(mutex);
        V(empty);        
    }
}
```

### 读者写者问题

```c++
semaphore rw = 1, mutex = 1; // mutex是对读者共享rw的限制
int cnt = 0;

void writer(){
	while (1){
		P(rw);
        写
        V(rw);
    }
}

void reader(){
    while (1){
        P(mutex);
        if (cnt == 0) P(rw); // 如果此时是第一个读者，申请读写控制
        cnt ++;
        V(mutex);
        
        读;
           
        P(mutex);
        cnt --;
        if (cnt == 0) V(rw); // 如果没有读者了，松开读写控制
        V(mutex);
    }
}
```
