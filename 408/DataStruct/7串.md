### 1.2 kmp

**定义：**

-   p 是 pattern 字符串, t 是 text 字符串

-   next[j]：p[0, j - 1]这个段最长的、公共的、前后缀的长度

**创建next代码**

-   t,j 在进入if的正确条件的时候，一定保持着p[0 ~ t-1] == p[xxx ~ j-1]的前后缀相等

```c
int t = -1;
next[0] = -1;

while (j < m - 1){
    if (t < 0 || p[t] == p[j]){
        t ++, j ++;
        next[j] = t;
    }else{
        t = next[t];
    }
}
```

**匹配代码：**

```c
while (j < m && (i - j) <= (n - m)){
    if (j > 0 || t[i] == t[j]){
        i ++, j ++;
    }else{
        j = next[j];
    }
}

return i - j;

```

改良版本`next`的把字符相同且存在 `next` 指向关系的节点用并查集 `merge` 起来，最后通过 `next[fa[i]]` 实现路径压缩，运行结果完全一致。

`next[i]`和`i`合并规则如下
$$
next[i] \ge 0 \space and \space p(i) = p(next[i])
$$
